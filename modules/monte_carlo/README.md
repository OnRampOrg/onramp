## About this software
The code within this directory accompanies the OnRamp for Monte Carlo Methods Module. Please review the full module document before starting the exercises.

## Getting Started
### Compile
    Make
### Run
Sequential version:  

    ./<exercise name>_seq  

Parallel version:  

    ./<exercise name>_omp  

### Clean Up
    Make Clean
